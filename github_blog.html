Github上的静态网页

参考了阮一峰博客上的这篇搭建一个免费的，无限流量的Blog----github Pages和Jekyll入门。我使用github Pages这种方法，网页的主题使用github现有的主题，并在上面修改和添加内容。

SyntaxHighlighter代码高亮

一个初级coder的博客上面也不会有什么东西，暂时能用到的就是一个代码高亮插件了。

我用的是SyntaxHighlighter，版本号3.0.83。在头部加上如下的文件和js函数

<script type="text/javascript" src="scripts/shCore.js">
<script type="text/javascript" src="scripts/shBrushJava.js">
<link type="text/css" rel="stylesheet" href="styles/shCoreDefault.css"/>
<script type="text/javascript">SyntaxHighlighter.all();

在引用的代码处加上

<script type="syntaxhighlighter" class="brush: java;"></script>

以上class的参数表示高亮java的代码，若是C++，则需要加上

<script type="text/javascript" src="scripts/shBrushCpp.js">

相应的，引用代码时，用的class需要改成

<script type="syntaxhighlighter" class="brush: c++;">

与java和c++类似，其他语言也有相应的语法高亮文件可供使用
